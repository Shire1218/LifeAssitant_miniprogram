# front 目录说明

此目录用于存放前端相关代码（如 H5 页面、Web 管理后台等）。

当前项目为纯微信小程序项目，暂无前端 Web 代码。
后续如需开发 Web 端管理后台或 H5 页面，可在此目录下创建。

## 推荐结构
```
front/
├── public/          # 静态资源
├── src/
│   ├── components/  # 公共组件
│   ├── views/       # 页面视图
│   ├── router/      # 路由配置
│   ├── store/       # 状态管理
│   ├── utils/       # 工具函数
│   └── App.vue      # 根组件
├── package.json
└── vite.config.js
```
