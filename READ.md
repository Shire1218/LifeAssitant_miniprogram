# 生活助手小程序

一个面向个人生活管理的小程序，包含记账、日程管理、饮食管理、生活随笔等功能。

## 项目结构

```
LifeAssitant_miniprogram/
├── miniprogram/          # 微信小程序代码
│   ├── pages/            # 页面
│   │   ├── bill/         # 记账模块
│   │   ├── schedule/     # 日程管理模块
│   │   ├── diet/         # 饮食管理模块
│   │   └── note/         # 生活随笔模块
│   ├── components/       # 公共组件
│   ├── utils/            # 工具函数
│   ├── static/           # 静态资源
│   ├── app.js            # 小程序入口
│   ├── app.json          # 小程序配置
│   └── app.wxss          # 全局样式
├── front/                # 前端代码（预留）
├── backend/              # 后端代码（预留）
├── LICENSE
└── README.md
```

## 功能模块

### 1. 记账模块
- 支持收入/支出两种类型
- 内置多种分类标签
- 月度统计（总收入、总支出、结余）
- 按日期分组展示
- 支持新增、编辑、删除、批量删除
- 自定义日期和备注

### 2. 日程管理模块
- 月视图日历，日程日期打点标记
- 支持标题、时间段、地点、备注
- 消息提醒配置
- 标记日程完成（完成后加删除线）
- 按时间正序排列

### 3. 饮食管理模块
- 区分早餐/午餐/晚餐/加餐
- 支持填写饮食详情、上传图片
- 快捷点餐（内置常见饮食模板）
- 今日饮食概览

### 4. 生活随笔模块
- 富文本编辑（加粗、斜体、下划线、颜色、对齐、列表等）
- 支持插入图片
- 自动草稿保存
- 列表展示（标题、摘要、时间、首图预览）
- 详情预览

## 技术栈

- **核心技术**：微信原生 WXML + WXSS + ES6+ JavaScript
- **UI 组件**：微信官方 WeUI 组件库
- **数据存储**：微信原生 Storage 本地持久化存储
- **富文本**：微信官方原生 editor 富文本组件

## 快速开始

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，选择"导入项目"
3. 选择 `miniprogram` 目录
4. 填写 AppID（可使用测试号）
5. 点击"导入"即可运行
